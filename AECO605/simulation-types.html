Types of
Simulation
1 of 3
Static vs. Dynamic
Static:
Represents a system at a fixed point in time — time plays no role.
Monte Carlo models (e.g., estimating π by random sampling)
Dynamic:
Represents a system as it evolves over time.
Conveyor belt system in a factory
2 of 3
Deterministic vs. Stochastic
Deterministic:
No random components; output is fully determined by inputs.
System of differential equations for a chemical reaction
Stochastic:
Contains random inputs; output is itself random — an estimate only.
Queueing & inventory systems
3 of 3
Continuous vs. Discrete
Discrete:
State changes at specific event times.
Traffic flow tracking individual cars
Continuous:
State changes continuously; described by differential equations.
Aggregate traffic flow model
